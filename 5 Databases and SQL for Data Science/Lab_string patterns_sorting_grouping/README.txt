The practice problems for this Lab will provide hands on experience with string
patterns, sorting result sets and grouping result sets, and how to run
SQL scripts to create several tables at once, as well as how to load data into tables
from .csv files.

Db2 database service on IBM Cloud will be used to work on a sample HR database for this Lab. This HR database
schema consists of 5 tables called EMPLOYEES, JOB_HISTORY, JOBS,
DEPARTMENTS and LOCATIONS. Each table has a few rows of sample data The
following diagram shows the tables for the HR database.

There are three parts to this lab:
I. Creating tables
II. Loading data into tables
III. Composing and running queries
